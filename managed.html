Managed Server

XMPP hosting by JSXC

Beta service! This service is currently in testing stage and only available for OJSXC >= 3.3.0.

You like to have your own full-blown XMPP server, without to hassle with the installation and maintenance of a server? Good news for you: JSXC will host your server for free*.

You can get it in 4 easy steps:

  • Go to the Nextcloud app store and install the "JavaScript XMPP Chat"
  • Open your admin page
  • Select "Managed"
  • Click on "Register" and your done

* up to 20 users in the testing stage

FAQ

Frequently Asked Questions

Is it a real XMPP server?

Yes. You will get your own virtual host on our Ejabberd installation.

Which XMPP domain will I get?

We will use your Nextcloud domain as subdomain of jsxc.ch. For example if use cloud.mydomain.com for your Nextcloud installation, we will use cloud.mydomain.com.jsxc.ch as your XMPP domain.

Can I use other clients?

As we said, you will get a full-blown XMPP server and this means you can use every XMPP client. Try Pidgin, Gajim or Conversations. Tip: Use app passwords.

Is this save?

Sure. We don't have access to your Nextcloud data and all XMPP connections from JSXC will use time-limited tokens, so we don't get your password. If you use a third-party client we recommend to use app passwords. You should also make sure, that no one else is able to install another Nextcloud instance on your host, because in this case he can overwrite your registration.

Is this federated and what the hack is federated?

Federated means you are able to communicate with every other XMPP user if they support server-to-server communication. It's like E-Mail and yes, our XMPP server supports this.

Are there really no constraints?

Ok, you caught us. Our service doesn't work if you use E-Mail addresses as user ids in Nextcloud, but we are working on this.
