Integration into your own webapp

Other Integration

JSXC can be integrated in other web pages as well. If you want to add comfort features such as unified login, configuration, or JID detection, you need to add specific code to do this (don't worry, we're working on a how-to description). On the right, you can see an integration into the University's Typo3 Web Content Management System.

Besides the standard features of JSXC, you can optionally add the following:

  • Unified login: Logging into your app also logs you in to JSXC/XMPP
  • Configuration from within your app's administrative interface
  • Recognizing JID, making it active and adding presence information to it
  • Automatically join MUCs associated with this page (e.g. for group documents)
